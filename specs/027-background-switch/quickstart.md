# Quickstart: Background Switcher

## Goal

Allow players to toggle the background between kommun and danger using the B key while keeping gameplay uninterrupted and remembering the last selection on the same device.

## Steps

1. Open the game in a browser.
2. Press the B key to toggle the background.
3. Confirm the background updates immediately and gameplay continues.
4. Reload the page and confirm the last selected background is preselected.
5. Verify there is no on-screen background selector, only an indicator of the active background.
