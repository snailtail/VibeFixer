# Quickstart: Active Session List

## Goal

Verify that admins can view an up-to-date list of active sessions with required details.

## Prerequisites

- App running locally.
- Admin login available.

## Steps

1. Start the backend and frontend.
2. Open the game in two separate browser tabs and start a session in each.
3. Log in to the admin console.
4. Navigate to the active session list page.
5. Confirm each active session appears with identifier, status, start time, elapsed duration, score, remaining artifacts, and player speed modifier.
6. End one session and confirm it disappears from the list within 4 seconds.
7. End all sessions and confirm the empty-state message is displayed.
