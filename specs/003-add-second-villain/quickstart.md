# Quickstart: Second Vibe Coder + Toast

## Prerequisites

- Node.js 20+

## Local Development

1. Start the backend server: `node backend/src/server.js`.
2. Open `http://localhost:3000/`.

## Smoke Tests

- Start a round and verify coder speed stays constant before and after 30 seconds.
- Observe a toast warning shortly before 30 seconds while gameplay continues.
- Confirm a second coder appears at ~30 seconds and drop frequency doubles.
