# Quickstart: FOMO Demon, Cast, and Event Log

## Prerequisites

- Node.js 20+

## Local Development

1. Start the backend server: `node /Users/magnus/src/Work/VibeFixer/backend/src/server.js`.
2. Open `http://localhost:3000/` in a modern desktop browser.

## Smoke Tests

- End a round with zero unchecked code blocks and confirm the defeated FOMO Demon message displays.
- End a round with remaining blocks and confirm the enraged image and sound display at game over.
- Trigger Imp Ediment and second coder events and confirm log entries appear on the left panel.
- Deposit artifacts and break blocks, confirming log entries and blocker count updates.
- Wait for vibe coder drops to confirm log entries for new unchecked code.
- Scroll below the game area and confirm cast and credits sections render.
